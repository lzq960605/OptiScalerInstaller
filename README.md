# OptiScaler Installer for Intel LunarLake

**1. Supported Device**
   - MSI Claw 8 AI+ (Intel LunarLake)
   - MSI Claw 7 AI+ (Intel LunarLake)
   - MSI Claw (Methor Lake)

**2. Supported Features**
   - Install OptiScaler to the selected game directory
   - Install pre-configured OptiScaler INI settings
   - in-game graphics optimization for OptiScaler (selected games only)
   - OptiPatcher installation for supported games
   - Reframework installation for supported games
   - Unreal Engine 5 GI bug fix (via DXGI Proxy patch, selected games only)

**3. Supported Games**
   - The list of supported games will appear when the installer launches.



**References**
 - OptiScaler : https://github.com/optiscaler/OptiScaler
 - OptiPatcher : https://github.com/optiscaler/OptiPatcher

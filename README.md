# OptiScaler Installer


## Requirements

- Internet connection needed (App needs to load game DB from Google Sheet)
- Optiscaler_0.9.0-pre11 (20260311)_RC4.5.7z or later version
  > **Github version not supported**
  > **Please download the latest from OptiScaler discord**

## Supported GPU

- Intel Lunarlake, Methor Lake
- AMD 780M, 890M, 8060S(Al Max+ 395)
     > Default installation option is XeSS + XeFG
     > So if you want to change to FSR, you can do it on OptiScaler overlay (press Insert key)

## Supported Features

 - Control supported GPU (Live)
   > Live means no App update required. App will load information from Google Sheet
 - Automatic game scan at launch (C: D: E: drive Steam library folder)
   > You can select game folder manually
 - Install OptiScaler files & setting(ini) for specific game (Live)
	> **No need to set additional OptiScaler option, Just install and run the game.**
 - In-game graphic option optimization for OptiScaler (for selected games only) (Live)
 - Automatic installation for additional module (for selected games only)
   > OptiPatcher, Reframework, Unreal5 G/I bug fix patch
 - User notification feature (pop-up, information window) (Live)
 - RTSS option change guide pop-up (if needed. no pop-up with proper RTSS option)
 - OptiScaler download link, the latest OptiScaler version info. (Live)

## Suppoted Games

 - Supported game list pop-up will be shown at App launch (Live)

## How to install
  - App will scan pre-defined game folder (C/D/E drive Steam library)
    > If game is not displayed, select game folder and scan manually
  - App will show notice and supported game list
  - Select a game to install
  - Check popup or information which provided before install
  - Select OptiScaler archive file
    > download the latest archive and use as is without extracting.
  - Click install button
  - after install completed, run a game and change in-game upscaling & frame generation option (follow the information)
  - Re-launch the game and check FPS
    > Press insert key to see OptiScaler overlay
